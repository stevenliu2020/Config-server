Git常用命令
1 创建目录 D:\WorkSpaceGit\demo  右键，选择Git Bash Here
2 输入 git clone http://192.168.110.89/airsaleSource/ch-gds-demo.git 回车
3  $cd ch-gds-demo
4  $git checkout ‘develop’  
5  $git pull  拉取默认分支最新代码并合并，相当与git fetch和git merge
6  $git status  查看本地代码状态
7  $echo ‘test’ >> Test.html   创建新文件
8  $git add Test.html  添加文件到暂存区
9  提交 $git commit –m ‘添加test’
        $git push origin develop
此时http://192.168.110.89/airsaleSource/ch-gds-demo/tree/develop 有Test.html文件
10 创建分支 $git branch release1.0.0.0
            $git push origin release1.0.0.0 
11 删除分支 $git branch –d release1.0.0.0
   删除远程分支 $git push origin --delete release1.0.0.0

12 删除远程仓库的.idea 目录
$git rm –r -–cached .idea
$git commit –m ‘删除目录’
$git push
13 增加  $git add .
14 合并master  注意这里是全部合并
   $git checkout master
   $git pull 
   $git merge origin/develop   #远程仓库合并
当然也可以 $git merge develop    #本地develop合并到本地master 
              $git push origin master 
16 合并一次提交
   $git checkout develop
   $git reflog   #查看提交记录
   $git checkout master 
   $git cherry-pick xxxxxxx  提交记录中的7位版本号
   $git push
